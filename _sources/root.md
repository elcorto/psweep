**psweep**

Welcome to the `psweep` docs. We hope that you'll soon loop like us!

```{tableofcontents}
```

**Install**

```sh
$ pip install psweep
```

Dev install of this repo:

```sh
$ pip install -e .
```

See also <https://github.com/elcorto/samplepkg>.

**Tests**

```sh
# Run in parallel, e.g. use 4 cores
$ pytest -vx -n4
```
