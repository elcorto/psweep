psweep.psweep.seq2dicts¶

psweep.psweep.seq2dicts(name: str, seq: Sequence[Any])¶

Create a list of single-item dicts holding the parameter name and a value.

>>> plist('a', [1,2,3])
[{'a': 1}, {'a': 2}, {'a': 3}]
