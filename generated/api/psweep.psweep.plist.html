psweep.psweep.plist¶

psweep.psweep.plist(name: str, seq: Sequence[Any])[source]¶

Create a list of single-item dicts holding the parameter name and a value.

>>> plist('a', [1,2,3])
[{'a': 1}, {'a': 2}, {'a': 3}]
